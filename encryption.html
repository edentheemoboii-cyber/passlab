Encryption and Decryption

Purpose

We used simple ciphers to understand how encryption changes readable text into unreadable form. We then attacked the ciphers to see whether the original message could be recovered.

Caesar Cipher

Each letter is shifted by a fixed number.

HELLO + shift 3 = KHOOR

Attack: Try all 25 possible shifts.

Substitution Cipher

Each letter is replaced by another fixed letter.

A→Q, B→W, C→E

Attack: Use frequency analysis on longer text.

XOR Cipher

Characters are converted into binary and combined with a key.

H XOR key = encrypted value

Attack: Try all short key values.
