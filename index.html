Are Your Passwords Safe?

Introduction

Due to the increasing reliance on digital programs, the need for safety and protection of private data has also increased. Cyber threats like unauthorized access and weak passwords, expose users to various risks. Our project will focus on basic cyber security techniques, specifically file encryption and password strengthening analysis, so we can understand how data can be stored effectively.

We tested common passwords, memorable passwords, random passwords, and passphrases against dictionary attacks, brute-force attacks, and simple encryption methods.

This website shares our experiment process, evidence, results, and practical recommendations for safer personal digital security.

Scope of Study

  • Understanding how encryption protects data.
  • Testing password strength.
  • Analyzing password weaknesses against dictionary and brute force attacks.
  • Recommending ways to improve personal digital security

Some Study Questions

  1. How quickly can common passwords be cracked using a dictionary attack?
  2. How does password length affect brute-force cracking time?
  3. Which simple cipher is easiest to break?
  4. What password habits should normal users follow?

Methodology

1. Dictionary Attack

  • Collect common passwords from NordPass
  • Write the code that could perform the dictionary attack
  • Test the weak, strong, random, and memorable passwords
  • Record carcking time and success rate
  • Analyse which passwords are most resistant

2. Brute Force Attack

  • Generate passwords of different lengths
  • Test numbers-only and alphanumeric passwords
  • Develop a python brute-force attack program.
  • record the time required to crack passwords.
  • Analyze the effect of password length on security.

3. Encryption and Decryption

  • Encrypt data using Caesar Cipher, Substitution Cipher, and XOR Cipher.
  • Attempt to recover the original data using suitable attacks.
  • Record the time require to decrypt the data.
  • Evaluate the effectiveness of each encryption method.

4. Final Analysis

  • Compare all the results from the dictionary attack, brute-force attack, and encryption tests to determine the most secure passwords and encryption methods.
